# Análisis de Desempeño de Tiendas

## Descripción
Este proyecto consiste en un único archivo de Google Colab cuyo objetivo es poner en práctica los contenidos aprendidos en el curso mediante un proyecto simple de análisis de datos.  
Se analiza un conjunto de tiendas con la finalidad de identificar la tienda con peor desempeño general para evaluar su venta.

---

## Propósito del análisis
El propósito principal del análisis es:
- Evaluar el desempeño de cada tienda usando métricas básicas (ventas, categorías, etc.)
- Comparar las tiendas entre sí
- Determinar cuál presenta el peor rendimiento global
